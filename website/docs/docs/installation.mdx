# Installation

Install the component via [npm](https://www.npmjs.com/):

```bash
npm install intelli-chat
```

For React, install the following instead: <br />

```bash
npm install intelli-chat-react
```

Access the component via CDN:

```
https://unpkg.com/intelli-chat@1.4.11/dist/intelliChat.bundle.js
```
